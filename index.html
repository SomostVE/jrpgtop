Comment utiliser JRPGTop

  • Choisis le classement

    Dans le menu de droite, choisis Toutes les licences ou Tous les jeux ou sélectionne directement une licence.

  • Choisis le thème

    Dans le menu de gauche, utilise Thème pour choisir le fond et les couleurs du classement.

  • Classe ton top

    Fais glisser les cartes ou saisis directement leur numéro de classement.

  • Finalise et exporte

    Utilise Finaliser le top pour masquer les éléments non classés. Ouvre Aperçu final puis choisis Exporter en PNG pour enregistrer l’image.
